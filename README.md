# NumNum

命令行交互脚本 `uv run generator.py` 可以批量生成 100 以内的加减法题目，并支持导出适合 A4 打印的网页。

## 使用步骤
1. 运行 `python generator.py`，按照提示选择题型、题量、加减法比例以及难度区间。
2. 生成的题目会立即显示在终端。
3. 如果需要打印，回答 `y` 以生成网页，并输入（或直接回车使用默认）输出文件名，例如 `worksheet_2024-03-20.html`。
4. 用浏览器打开生成的 HTML 文件，设置纸张为 A4 并打印即可。

## 网页排版说明
- 页面使用两列栅格布局，适合直接打印到 A4 纸张。
- 页眉包含题型、题量和难度区间，同时预留“姓名 / 日期”填写区域。
- 每道题在行尾预留横线，方便学员填写答案。
- 同一个答案默认最多出现 10% 的题量，且两数运算会自动保证加减各占 50%。
